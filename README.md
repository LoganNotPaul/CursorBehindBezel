# CursorBehindBezel
A tool specifically designed for the border resistance feature of LittleBigMouse (though you don't necessarily need it).

## What it does
It hides the mouse cursor along the edge of the screen so it doesn't drag awkwardly across the screen when moving your cursor to another monitor. 

It's fully customizable: you can adjust where on any monitor and on which edges the cursor will become invisible. You can also adjust the height and width of the zones where the cursor disappears.

## How to use
- Download and run `CursorBehindBezel.exe` (or run the source `.ahk` script using AutoHotkey v2).
- **(Important)** To customize where the cursor disappears, press `Ctrl + Shift + E`. If you already have that hotkey set to something, well fuck you.
- It runs in the system tray. To close, click on the system tray, right-click the app, and click **Exit**.

## Compatibility
Should work with any combination of any size screens and any number of screens. It even works with mismatched or different size screens.

## Why?
Because it gives the illusion that the mouse is going behind the screen instead of dragging on the edge until it goes to the other screen. Which, in my opinion, is a much better way to use the border resistance feature.
